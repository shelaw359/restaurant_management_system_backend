README.md — Restaurant Management System (Backend)
Project Overview

A full‑stack Restaurant Management System built to manage core restaurant operations such as menu handling, orders, reservations, and user roles. The backend is developed with NestJS, TypeORM, and MS SQL, while the frontend is built with React, TypeScript, and Tailwind CSS.

This project demonstrates real‑world backend architecture, RESTful API design, and full‑stack integration.

Installation & Setup
# Clone the backend repository
git clone https://github.com/shelaw359/restaurant_management_system_backend.git
cd restaurant_management_system_backend


# Install dependencies
pnpm install


# Run development server
pnpm run start:dev

Ensure MS SQL is running and environment variables are configured correctly.
